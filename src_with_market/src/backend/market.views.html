Brief:
    REST API endpoints (ViewSets) for marketplace resources using DRF.
    Provides read-only categories, read/write products & listings, and
    authenticated buyer order creation + retrieval.
 
Important endpoints:
    - GET /api/market/health/  -> service liveness probe
    - /api/market/categories/  -> list/retrieve
    - /api/market/products/    -> list/create/retrieve/update/delete (auth for write)
    - /api/market/listings/    -> list/create/retrieve/update/delete (auth for write)
    - /api/market/seller/me/   -> current user's SellerProfile
    - /api/market/orders/      -> buyer's list/create/retrieve (auth)
